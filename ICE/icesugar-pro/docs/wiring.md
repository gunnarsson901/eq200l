# ICESUGAR-PRO BRB ↔ LAN8720 Ethernet Board

## P2 → ETH1 Connection
| P2 Pin | ETH1 Pin | Signal |
|:------:|:--------:|:------:|
| R1     | NC       | -      |
| P1     | TXD      | TX Data |
| N1     | RX1      | RX Data 1 |
| M1     | CRS      | Carrier Sense |
| K1     | MDC      | Management Data Clock |
| GND    | GND      | Ground  |
| 3V3    | VCC      | Power   |
| P2     | TX1      | TX Data 1 |
| N3     | TX_en    | TX Enable |
| M2     | RXD      | RX Data |
| L2     | nINT/RETCLK | Interrupt/Ref Clock |
| G2     | MDIO     | Management Data I/O |

## P3 → ETH2 Connection
| P3 Pin | ETH2 Pin | Signal |
|:------:|:--------:|:------:|
| B4     | NC       | -      |
| B5     | TXD      | TX Data |
| B6     | RX1      | RX Data 1 |
| B7     | CRS      | Carrier Sense |
| B8     | MDC      | Management Data Clock |
| GND    | GND      | Ground  |
| 3V3    | VCC      | Power   |
| A4     | TX1      | TX Data 1 |
| A5     | TX_en    | TX Enable |
| A6     | RXD      | RX Data |
| A7     | nINT/RETCLK | Interrupt/Ref Clock |
| A8     | MDIO     | Management Data I/O |
